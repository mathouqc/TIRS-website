# TIRS-website
Repository for the content used to build the TIRS website: thatisrocketscience.com.

Minimal prototype based on Astro, MDX, and a Three.js component (react-three-fiber).

## Documentation

- [Cahier des charges](doc/CAHIER_DES_CHARGES.md)
- [Comment faire (HOW TO)](doc/HOW_TO.md)
- [Guide du développeur](doc/GUIDE_DU_DEVELOPPEUR.md)
- [TODO projet](doc/TODO.md)

Commands:

```bash
npm install
npm run dev
```

Deployment: The `.github/workflows/deploy.yml` workflow builds and publishes `./dist` to GitHub Pages.
